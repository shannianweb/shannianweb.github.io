首次安装安全提示

闪念是经过代码签名的正版应用。部分平台首次安装会触发系统保护提示，按下列步骤操作即可。

macOS

首次打开可能看到：
Apple 无法验证「闪念」是否包含恶意软件。
  1. 点击 取消，不要点「移到废纸篓」
  2. 打开 系统设置 → 隐私与安全性
  3. 在安全性区域找到已阻止提示
  4. 点击 仍要打开 并确认
  5. 之后再打开不会再提示
快捷方式：访达中右键应用 → 打开。

Android

可能提示：
禁止安装未知来源应用
  1. 在弹窗中点击 设置
  2. 开启允许来自此来源的应用
  3. 返回后点击 安装
说明：因未通过 Google Play 分发，属于正常安全提醒。

Windows

  1. 下载 Setup 安装版或 Portable 便携版
  2. 若 SmartScreen 提示，选择更多信息 → 仍要运行
  3. 按向导完成安装，或直接运行便携版
提示：便携版适合不想写入系统目录的用户。

Linux

  1. DEB：使用系统包管理器安装
  2. AppImage：赋予可执行权限后运行
  3. 当前提供 amd64 构建，Intel / AMD 通用
提示：若架构不匹配，请等待后续包或联系反馈。
